# Domino's Sales Analysis Dashboard

This project is a Power BI dashboard created for analyzing Domino's sales data.

## Features
- Total Sales Analysis
- Top Selling Pizzas
- Revenue Insights
- Order Trends
- Category-wise Performance
- Interactive Filters and Visualizations

## Tools Used
- Power BI
- Excel / CSV Dataset

## Project Purpose
The goal of this project is to practice data analytics and data visualization skills using Power BI
